This is a simple to use comic book reader that supports cbz, cbr and pdf. 
This is written in python and not feature complete. 

You can currently:
* Read a comic
* See the info about the comic
* View in single or double page
* Zoom in / Out

Coming to future versions:
* Library management
* Simple server to work with some old school BBS software to allow users to read comics from your file bases (Mystic BBS)
* Rating system
* RSS Feed access
